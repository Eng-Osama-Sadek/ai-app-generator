# AI App Generator 🚀 (Full-Stack Task)

This project is a dynamic application generator built with **React (Vite)**, **Node.js (Express)**, and **Prisma ORM**. It allows users to create database tables dynamically from the frontend.

## 🛠 Tech Stack
- **Frontend:** React.js, Axios, CSS3.
- **Backend:** Node.js, TypeScript, Express.
- **Database:** SQLite with Prisma ORM.
