# AI App Generator 🚀 (Full-Stack Dynamic Schema)

A sophisticated full-stack application that demonstrates **Dynamic Database Schema Generation**. Users can define table structures from the UI, which are then provisioned in real-time in the SQLite database using Prisma ORM.

## 🏗 System Architecture
- **Frontend:** Built with **React (Vite)** and **TypeScript**, utilizing **Axios** for asynchronous API communication.
- **Backend:** A **Node.js/Express** server that processes dynamic schema requests.
- **Database Layer:** Uses **Prisma ORM** to handle migrations and schema updates on-the-fly. 
- **Workflow:** UI Request -> Backend Processing -> Prisma Schema Update -> Database Migration -> Studio Visualization.

## 🛠 Tech Stack
- **Frontend:** React.js, TypeScript, Axios, CSS3.
- **Backend:** Node.js, Express, TypeScript.
- **Database:** SQLite with Prisma ORM.
